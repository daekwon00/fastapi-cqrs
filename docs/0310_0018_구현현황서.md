## 2. 아키텍처 매핑

| Spring Boot | FastAPI |
|---|---|
| Controller (`@RestController`) | Router (`APIRouter`) |
| Service (`@Service`) | Service class (Depends 주입) |
| MyBatis Mapper XML | Repository class (asyncpg raw SQL) |
| Java Record DTO | Pydantic v2 BaseModel |
| Spring Security Filter | FastAPI middleware + Depends |
| `@Value` / application.yaml | pydantic-settings `BaseSettings` + .env |
| HikariCP 커넥션 풀 | asyncpg.create_pool() |

### CQRS 패턴

```
Router → CommandService → CommandRepository (INSERT/UPDATE/DELETE)
Router → QueryService   → QueryRepository   (SELECT)
```

### 핵심 기술 변환 (적용 완료)

- MyBatis `#{param}` → asyncpg `$1, $2, ...` (positional params)
- MyBatis `<if>/<choose>` → Python 조건문으로 동적 쿼리 빌드
- MyBatis `<foreach>` IN절 → PostgreSQL `ANY($1::text[])`
- SseEmitter → sse-starlette `EventSourceResponse` + async generator
- BCrypt + 평문 패스워드 fallback 패턴 유지

## 3. 디렉토리 구조 (구현 완료)

```
fastapi-cqrs/
├── pyproject.toml / requirements.txt
├── .env.example
├── Dockerfile / docker-compose.yaml
├── app/
│   ├── main.py              # FastAPI 앱, lifespan, 미들웨어
│   ├── config.py             # pydantic-settings
│   ├── database.py           # asyncpg 풀 생성/해제
│   ├── common/               # ApiResponse, PageResponse, 예외, 페이지네이션
│   ├── security/             # JWT, 패스워드, 인증 Depends
│   ├── auth/                 # 로그인, 회원가입, 토큰갱신
│   ├── board/                # 게시판, 게시글 CRUD
│   ├── file/                 # 파일 업로드/다운로드
│   ├── user/                 # 프로필, 비밀번호 변경
│   ├── dashboard/            # 통계, 차트
│   ├── ai/                   # AI 채팅 SSE (Anthropic/OpenAI/Gemini)
│   ├── admin/                # 관리자 (user/board/dashboard)
│   └── system/               # 역할, 메뉴, 코드, 직급-역할
└── tests/
```

각 도메인 모듈 내부:
```
{module}/
├── router.py
├── schemas.py
├── service/
│   ├── command_service.py
│   └── query_service.py
└── repository/
    ├── command_repository.py
    └── query_repository.py
```

## 9. Cloud MSA 확장 고려사항

현재 모놀리식으로 구현 완료. 모듈별 독립 패키지 구조로 향후 분리 가능:
- **서비스 추출**: 각 도메인 모듈을 독립 FastAPI 마이크로서비스로 분리 가능
- **공유 커널**: `common/`, `security/`는 공유 라이브러리(pip 패키지)로 추출
- **API Gateway**: CORS를 게이트웨이 레벨로 이관
- **이벤트 드리븐**: 서비스 레이어에 이벤트 발행 포인트 추가 가능
- **컨테이너 오케스트레이션**: 각 모듈 Dockerfile → Kubernetes 배포
